ToolChain 만들기
=========

<br />

이제 여러 작업을 하나의 흐름으로 실행해보겠습니다.   
어시웍스에서는 여러 Tool을 연결해 실행하는 작업 흐름을 ‘ToolChain’이라고 부릅니다.

<br />

이번 가이드에서는 방금 만든 텍스트 요약 Tool을 ToolChain에 넣어 실행해봅니다.

<br />

## 1. ToolChain 생성하기

#### 1) 좌측 메뉴에서 'ToolChain'을 클릭합니다.
![툴체인 목록 페이지](/images/docs/getting-started/create-toolchain/1-1.png)

<br />

#### 2) '+ 생성' 버튼을 클릭합니다.
![툴체인 목록 페이지 : 생성 버튼 클릭](/images/docs/getting-started/create-toolchain/1-2.png)

<br />

#### 3) 이름 수정 버튼을 누르고, ToolChain 이름에 ‘텍스트 요약 체인’이라고 입력합니다.
![툴체인 생성 페이지 : 툴체인 이름/설명 작성 버튼 클릭](/images/docs/getting-started/create-toolchain/1-3(1).png)
![툴체인 생성 페이지 : 툴체인 이름/설명 작성창](/images/docs/getting-started/create-toolchain/1-3(2).png)

<br />

## 2. 입력 노드 추가하기
ToolChain은 실행에 필요한 값을 입력 노드에서 받습니다.

<br />

#### 1) '입력' 버튼을 클릭합니다.
![툴체인 생성 페이지 : 입력 노드 추가 버튼 클릭](/images/docs/getting-started/create-toolchain/2-1.png)

<Callout type="note">
<p className="text-gray-600">User Prompt는 언제든지 수정 가능하며, 실행 패널에서 바로 테스트할 수 있습니다.</p>
</Callout>

<br />

#### 2) 입력 이름에 ‘요약할 글’, 설명에 ‘요약할 원문 텍스트’라고 입력합니다.
![툴체인 생성 페이지 : 입력 노드 정보 입력창](/images/docs/getting-started/create-toolchain/2-2.png)

<br />

#### 3) 입력 노드가 잘 추가되었는지 확인해주세요.
![툴체인 생성 페이지 : 그래프에 입력 노드 추가됨](/images/docs/getting-started/create-toolchain/2-3.png)

<br />

## 3. 실행 노드에 Tool 연결하기
이제 ToolChain 안에서 실행할 Tool을 추가합니다.

<br />

#### 1) 노드 추가 영역 목록에서 ‘텍스트 요약’ Tool을 선택합니다.
![툴체인 생성 페이지 : 입력 노드 한개, 스텝 노드(Tool) 한개 있는 그래프](/images/docs/getting-started/create-toolchain/3-1.png)

<br />

#### 2) 입력 노드의 ‘요약할 글’을 텍스트 요약 Tool의 입력 필드에 연결합니다.
![툴체인 생성 페이지 : 입력 노드와 스텝 노드(Tool) 연결](/images/docs/getting-started/create-toolchain/3-2.png)

<br />

#### 3) 연결 선을 눌러 출력을 매핑해줍니다.
![툴체인 생성 페이지 : 연결선(엣지) 클릭. 연결된 노드끼리 입출력 매핑](/images/docs/getting-started/create-toolchain/3-3.png)

<br />

## 4. 출력 노드 추가하기

#### 1) ‘출력’ 버튼을 클릭합니다.
![툴체인 생성 페이지 : 출력 노드 추가 버튼 클릭](/images/docs/getting-started/create-toolchain/4-1.png)

<br />

#### 2) 출력 이름에 ‘요약 결과’라고 입력합니다.
![툴체인 생성 페이지 : 출력 노드 정보 입력창](/images/docs/getting-started/create-toolchain/4-2.png)

<br />

#### 3) 텍스트 요약 Tool의 출력값을 출력 노드에 연결합니다.
![툴체인 생성 페이지 : 스텝 노드(Tool)와 출력 노드 연결](/images/docs/getting-started/create-toolchain/4-3.png)

<br />

#### 4) 연결 선을 눌러 출력을 매핑해줍니다.
![툴체인 생성 페이지 : 연결한 스텝 노드와 출력 노드의 입출력 매핑](/images/docs/getting-started/create-toolchain/4-1.png)

<br />

## 5. 저장하고 실행하기

<Callout type="note">
<p className="text-gray-600">ToolChain은 저장되지 않은 변경사항이 있으면 실행할 수 없습니다. 실행 전에는 반드시 저장해주세요.</p>
</Callout>

<br />

#### 1) 저장 버튼을 눌러 ToolChain을 저장합니다.
![툴체인 생성 페이지 : 툴체인 저장 버튼 클릭](/images/docs/getting-started/create-toolchain/5-1.png)

<br />

#### 2) 전체 실행 버튼을 클릭합니다.
![툴체인 실행 페이지 : 전체 실행 버튼 클릭](/images/docs/getting-started/create-toolchain/5-2.png)

<br />

#### 3) 입력창에 아무 문장이나 입력한 뒤 실행합니다.
![툴체인 실행 페이지 : 전체 실행 입력값 작성 사이드바](/images/docs/getting-started/create-toolchain/5-3.png)

<br />

#### 4) 오른쪽에서 요약 결과를 확인합니다. 요약결과 옆 크게보기를 눌러 결과를 새 창으로 확인할 수 있습니다.
![툴체인 실행 페이지 : 전체 실행 결과 사이드바](/images/docs/getting-started/create-toolchain/5-4(1).png)
![툴체인 실행 페이지 : 결과값 크게보기](/images/docs/getting-started/create-toolchain/5-4(2).png)
